# Chengshuo Du Personal Homepage

Static GitHub Pages site for `https://chengshuodu.github.io/`.

## Publish

1. Create a public GitHub repository named `chengshuodu.github.io` under the GitHub account `chengshuodu`.
2. Upload these files to the root of that repository:
   - `index.html`
   - `.nojekyll`
   - `assets/css/styles.css`
   - `assets/img/profile-placeholder.svg`
3. In the repository settings, set GitHub Pages to deploy from the `main` branch and root folder if it is not enabled automatically.
4. Visit `https://chengshuodu.github.io/` after GitHub Pages finishes deployment.

## Edit Later

- Replace the placeholder text in `index.html`.
- Replace `assets/img/profile-placeholder.svg` with a real profile photo when ready.
- Keep `.nojekyll` so GitHub Pages serves the static files directly.
